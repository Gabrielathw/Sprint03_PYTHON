## Sobre o Projeto

O ChargeGrid Intelligence é uma proposta de solução para o ecossistema de carregamento inteligente de veículos elétricos da GoodWe, focada na expansão do ambiente residencial para o ambiente comercial

O projeto busca resolver desafios relacionados a:
- Controle inteligente de demanda energética;
- Interoperabilidade entre carregadores e plataformas;
- Tarifação e pagamento flexíveis;
- Experiência do usuário unificada;
- Integração com energia solar e inteligência artificial.
